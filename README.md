# SIGMA - Integrated Administrative Maintenance Management System

## Overview

SIGMA is a web-based management platform developed to support administrative maintenance operations, resource management, asset tracking, and scheduling processes within an institution.

The project was developed as part of an academic and real-world software engineering initiative, covering the complete software development lifecycle from requirements analysis to deployment.

## Key Features

* Asset and inventory management
* Maintenance scheduling
* Administrative resource tracking
* User management
* Reporting and monitoring tools
